              # vue展示3D模型

              # vue-3d-model

              ps：需求需要在网页上展示3d模型文件，学习three.js的成本太高，

              # 特点：

              • 组件可以在vue中展示模型，自动适应容器宽高
              • 支持多种文件格式（obj，json，fbx等）
              • 可以进行多种事件绑定交互（缩放，单击，移入，移出等）

              # Install

              using npm

              npm install vue-3d-model --save

              <script src="https://unpkg.com/vue-3d-model/dist/vue-3d-model.umd.js"></script>

              # 使用场景：

              # 本地文件：适合模型文件较少较小的项目，直接把文件放在项目中

              1、把模型文件集中存放在public文件夹中

              2、按照之前发方法引入之后使用

               <!-- 使用 -->
 <model-fbx src="static/BYQ.FBX" />
              1
              2

              注意：static前面不要加 ‘/’ ,不然打包之后可能无法找到文件

              # 网络请求：适合文件多且大的模型文件，存放服务器

              动态请求最好是再封装一个预览模型组件

              1、把路径设置成动态,可以使用props传入

              <model-fbx :src="modelSrc" />
              1

              2、针对可能存在网络慢或者模型大等情况，需要再添加一个加载等待动画，以vue+elementui为例

              添加加载事件@on-progress

               <model-fbx @on-progress="modelLoading" :src="modelSrc" />
              1

              methods:

              modelLoading(xhr) {
    //创建加载动画
    const loading = this.$loading({
        lock: true,
        text: "正在加载模型，请等待！",
        spinner: "el-icon-loading",
        background: "rgba(0, 0, 0, 0.7)",
        customClass: "modelLoadingClass" //自定义class
    });
    if (xhr.loaded == xhr.total) {
        //加载完成之后，结束加载动画
        loading.close();
    }
},

              # 注意事项：

              • 模型加载是黑色的，或者一半黑色一半正常，不显示模型材质等问题，添加灯光配置即可

              <model-fbx @on-progress="modelLoading" :lights="lights" :src="modelSrc" />
              1

              data:

              //根据实际情况自行配置
lights: [
    {
        type: "HemisphereLight",
        position: { x: 0, y: 1, z: 0 },
        skyColor: 0xffffff,
        // groundColor: 0xFF0000, 此代码为灯光后颜色
        intensity: 1
    },
    {
        type: "DirectionalLight",
        position: { x: 1, y: 1, z: -10 },
        skyColor: 0xffffff,
        // groundColor: 0xff0000, //此代码为灯光后颜色
        intensity: 0.8
    },
    {
        type: "DirectionalLight",
        position: { x: 1, y: 1, z: 1 },
        color: 0xffffff,
        intensity: 0.8
    }
]

              # Documents

              # props

              prop type default example
              src string - './exapmle.obj'
              width number - 300
              height number - 300
              position object { x: 0, y: 0, z: 0 } { x: 100, y: 20, z: -10 }
              rotation object { x: 0, y: 0, z: 0 } { x: Math.PI / 2, y: 0, z: - Math.PI / 4 }
              cameraPosition object { x: 0, y: 0, z: 0 } { x: 1, y: 2, z: -3 }
              cameraRotation object { x: 0, y: 0, z: 0 } { x: 3, y: 2, z: -1 }
              scale object { x: 1, y: 1, z: 1 } { x: 2, y: 2, z: 3 }
              lights array -
              backgroundColor number/string 0xffffff 0xffffff/'#f00'/'rgb(255,255,255)'
              backgroundAlpha number 1 0.5
              controlsOptions object - see OrbitControls Properties
              crossOrigin string anonymous anonymous/use-credentials
              requestHeader object - { 'Authorization: Bearer token' }
              outputEncoding number THREE.LinearEncoding see WebGLRenderer OutputEncoding
              glOptions object { antialias: true, alpha: true } see WebGLRenderer Parameters
